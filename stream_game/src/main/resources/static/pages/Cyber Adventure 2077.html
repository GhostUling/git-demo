Cyber Adventure 2077

《Cyber Adventure 2077》是一款开放世界动作冒险RPG游戏，由著名游戏工作室开发。游戏设定在一个黑暗未来的世界中，玩家将扮演一位冒险者，在这个拥有丰富科技和危险冒险的世界中生存并崭露头角。

主要特点：

  • 开放世界探索
  • 深度角色定制
  • 多样化的游戏剧情和结局
  • 先进的视觉特效和沉浸式体验
开放世界 角色扮演 动作冒险 科幻

系统需求

  • 操作系统: Windows 10 64-bit
  • 处理器: Intel Core i5-8400 / AMD Ryzen 3 3200G
  • 内存: 8 GB RAM
  • 显卡: NVIDIA GeForce GTX 970 / AMD Radeon RX 570
  • 存储空间: 70 GB 可用空间
