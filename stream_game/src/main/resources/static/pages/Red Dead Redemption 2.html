游戏介绍

亚瑟·摩根和范德林德帮众在美国逐渐现代化的大时代背景下，展开一场关于生存与忠诚的史诗冒险。包含单人剧情与线上模式，体验真实的西部生活。

特色系统

  • 荣誉系统：玩家的选择影响NPC互动与剧情走向
  • 生存模拟：核心生命值/体力/死神之眼系统
  • 马匹系统：马匹驯养与深度互动
  • 环境交互：超过200种动物与动态天气系统

系统需求

  • 操作系统: Windows 10 64位
  • 处理器: Intel Core i7-4770K / AMD Ryzen 5 1500X
  • 内存: 12 GB RAM
  • 显卡: NVIDIA GTX 1060 6GB / AMD RX 480 4GB
  • 存储空间: 需要 150 GB 可用空间
