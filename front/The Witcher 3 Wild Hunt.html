游戏介绍

您是利维亚的杰洛特，收钱办事的怪物杀手。您可以在眼前这片怪物横行、饱受战火摧残的土地上尽情探索。 您手上的委托？追踪预言之子——希里，一件足以改变世界面貌的活生生的武器。

特色系统

  • 游戏模式：游戏模式根据其难度从低到高分别为：“看剧情就好”、“想流点汗”、“想流点血”、“死而无憾”。
  • 战斗系统：游戏以即时类战斗为基础，游戏角色需要根据不同的敌人制定相应的应对策略。 面对人类和类人生物需要使用钢剑，对付怪物时则需要使用银剑。拔剑后即进入战斗状态，系统将自动锁定离主角最近的目标。 通过物理攻击、法印和相应的躲避方式来进行作战。
  • 任务系统：游戏中的任务分为主线任务和支线任务，当完成相应任务后归档为已完成或失败。 玩家除了可以追随主线任务的标记依序前往不同的地图外，也可以暂时放下眼前的任务去好好地探索目前所在的地图， 游戏中的每一张地图上都有着相当丰富的支线任务与隐藏地点等待玩家去探索、发掘与执行。

系统需求

  • 操作系统: 64-bit Windows 7 or 64-bit Windows 8 (8.1)
  • 处理器: Intel CPU Core i7 3770 3.4 GHz or AMD CPU AMD FX-8350 4 GHz
  • 内存: 8 GB RAM
  • 显卡: NVIDIA GPU GeForce GTX 770 or AMD GPU Radeon R9 290
  • 存储空间: 需要 40 GB 可用空间
